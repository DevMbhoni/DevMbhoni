## 🚀 Featured Project

### 🎓 Student Success Predictor
> An intelligent full-stack platform that predicts student academic risk using machine learning — helping universities intervene before failure occurs.

**[🌐 Live Demo](https://student-success-predictor-beige.vercel.app) · [📂 Repository](https://github.com/DevMbhoni/student-success-predictor)**

| Layer | Tech |
|---|---|
| Frontend | React + TypeScript + TailwindCSS |
| Backend | ASP.NET Core 9 — Clean Architecture |
| ML Service | Python + FastAPI + Scikit-learn |
| Database | PostgreSQL (Neon) |
| Deployment | Vercel + Render + Docker |

**What it does:**
- 🔮 Predicts pass/fail probability and risk level (Low/Medium/High) per student per module
- 🤖 Three ML models: Logistic Regression, Random Forest, Decision Tree — ~97% accuracy
- 📊 Role-based dashboards for Students, Lecturers, Academic Advisors, and Administrators
- 💡 Explainable AI recommendations — tells you *why* a student is at risk and what to do about it
- 🔐 JWT authentication with role-based access control
